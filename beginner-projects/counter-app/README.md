## 🔢 Counter App

A responsive counting application built to master state tracking and user interaction fundamentals.

---

### 🚀 Features

- ➕ Increment and decrement counts
- 🔄 Instant state reset button
- 🎨 Visual UI feedback based on positive or negative value states

---

### 📚 What I Learned

- Event handling with .addEventListener() to capture core user interactions
- Safely tracking, mutating, and maintaining numerical state in JavaScript
- Conditionally manipulating elements and updating text nodes dynamically

---

### 🛠️ Tech Used

- HTML
- CSS
- JavaScript
