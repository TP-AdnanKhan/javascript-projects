## 🕒 Digital Clock

A real-time digital clock that displays the user's current local time.

---

### 🚀 Features

- 🕒 Live, synchronized local time tracking
- 📅 AM/PM standard 12-hour format conversion
- 📱 Responsive and minimalist display

---

### 📚 What I Learned

- Working with built-in JavaScript constructors like new Date()
- Managing continuous UI execution and state updates using setInterval
- Conditional formatting logic to handle standard time displays

---

### 🛠️ Tech Used

- HTML
- CSS
- JavaScript
