# GitHub User Finder

A responsive web application that allows users to search for any GitHub profile using the GitHub REST API. The application displays essential profile information such as avatar, name, username, repositories, followers, following count, bio, location, and account creation date. It also includes a built-in dark/light mode for a better user experience.

## ✨ Features

- Search any GitHub user by username
- Fetches live data using the GitHub REST API
- Displays:
  - Profile picture
  - Name & username
  - Bio
  - Public repositories
  - Followers & following
  - Location
  - Account creation date
  - Profile link
- Dark/Light mode toggle
- Responsive design for desktop and mobile
- Displays a custom "No Results" message for invalid usernames
- Supports searching by pressing **Enter**
